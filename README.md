## 🚀 快速开始

### 安装

```bash
npx clawhub@latest install shit-person
```

### 使用

直接向 AI 发送以下内容：

```
用 SHIT 狗屎人模式帮我看看这个项目：
[你的商业计划/文案/决策]
```

或者：

```
开启毒舌模式，分析一下这个 BP...
```

## 📤 输出格式

SHIT 狗屎人严格按照四个模块输出：

1. **【狗屎人的初步鉴定】** - 一句极具嘲讽意味的开场白
2. **【画饼粉碎机】** - 提取 2-3 个虚假繁荣词汇并毒舌翻译
3. **【致命毒点排查】** - 列出 3-5 个最可能导致失败的底层漏洞
4. **【最后的"构石"建议】** - 极度务实的行动建议（通常是劝退）

## 💬 使用示例

### 示例：商业计划书

**输入**：
> 我们计划做一个 AI 驱动的社交平台，通过算法匹配志同道合的用户，打造年轻人专属的元宇宙社区。预计第一年获取 100 万用户，三年内实现盈利...

**输出**：
```
【狗屎人的初步鉴定】
又来一个"改变世界"的，上次我见到这种 BP 还是在 2021 年，那家公司现在已经在仲裁庭上了。

【画饼粉碎机】
| 原文 | 毒舌翻译 |
|------|---------|
| "AI 驱动的社交平台" | "套个 AI 的壳，好拿融资" |
| "元宇宙社区" | "炒概念，实际就是个论坛" |

【致命毒点排查】
- 获客成本：100 万用户？你知道现在社交类 APP 的获客成本是多少吗？
- 竞争格局：微信、抖音、小红书把用户时间瓜分完了，你凭什么？
- 盈利模式：三年盈利？社交产品要么靠广告（你没流量），要么靠增值服务（你没付费意愿）

【最后的"构石"建议】
建议：放弃。沉没成本：你目前投入的时间约 200 小时，按你时薪 200 块算，已经亏了 4 万块。
```
